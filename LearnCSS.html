CSS Lessons

CSS is all about presentation. Learn the fundamentals step-by-step with practical examples and exercises.

1. What is CSS?

CSS stands for Cascading Style Sheets. It describes how HTML elements are displayed.

With CSS you can control colors, fonts, margins, borders, backgrounds, and layout.

/* Example: make all paragraphs blue */ p { color: blue; font-family: Arial, sans-serif; }

Exercise

غيّر لون نص الفقرات في صفحتك إلى لون مختلف، وجرّب تغيير نوع الخط والحجم.

2. Ways to Add CSS

You can add CSS in three ways: inline, internal, and external.

<!-- Inline --> <p style="color:red">Hello</p> <!-- Internal --> <style> p {{ color: green; }} </style> <!-- External --> <link rel="stylesheet" href="style.css">

Exercise

إنشئ ملف style.css مستقل، واربطه بصفحتك باستخدام <link> ثم اكتب داخله بعض التنسيقات.

3. Selectors

Selectors tell CSS which elements to style: by tag, class, id, or attribute.

/* Tag selector */ p {{ color: #e5e7eb; }} /* Class selector */ .card {{ background:#111827; }} /* ID selector */ #main-title {{ font-size: 32px; }}

Exercise

أضف كلاس باسم box لبعض العناصر، واكتب CSS يضيف لها خلفية وحدود.

4. Colors & Backgrounds

Colors can be written by name, HEX, RGB, or HSL. Backgrounds can be solid, gradient, or images.

body {{ background: #0f172a; }} .hero {{ background: linear-gradient(135deg, #1d4ed8, #f97316); }}

Exercise

غيّر خلفية الهيدر إلى تدرّج لوني (gradient) واختبر أكثر من لون حتى تحصل على شكل جميل.

5. Box Model (Margin, Border, Padding)

Every element is a box with content, padding, border, and margin.

.card {{ padding: 16px; border: 1px solid #4b5563; margin: 12px 0; }}

Exercise

طبّق box model على كرت واحد في الصفحة، وغيّر القيم حتى تفهم الفرق بين margin و padding.

6. Fonts & Text

Use CSS to control font family, size, weight, line-height, and text alignment.

h1 {{ font-family: "Tahoma", sans-serif; font-size: 32px; text-align: center; }} p {{ line-height: 1.8; }}

Exercise

غيّر خط العناوين إلى خط مختلف عن الفقرات، واجعل النص في الفوتر في المنتصف.

7. Display & Inline / Block

The display property controls how an element behaves (block, inline, inline-block, flex, etc.).

nav a {{ display: inline-block; padding: 8px 12px; }}

Exercise

حوّل عناصر القائمة في الـ navbar إلى inline-block ليكون بينها مسافة متساوية.

8. Flexbox Basics

Flexbox makes it easy to create flexible layouts that adapt to different screens.

.container {{ display: flex; gap: 16px; }} .sidebar {{ flex: 1; }} .content {{ flex: 3; }}

Exercise

قسّم الصفحة إلى عمودين: قائمة جانبية ومحتوى رئيسي باستخدام flexbox.

9. CSS Grid Basics

Grid is powerful for building two-dimensional layouts with rows and columns.

.grid {{ display: grid; grid-template-columns: repeat(3, 1fr); gap: 12px; }}

Exercise

أنشئ شبكة صور (3 أعمدة) لعرض لقطات شاشة من مشروعك.

10. Positioning

Use position (static, relative, absolute, fixed, sticky) to control where elements appear.

.badge {{ position: absolute; top: 8px; right: 8px; background:#f97316; padding:4px 8px; }}

Exercise

ضع شارة "New" في زاوية صورة باستخدام position: absolute مع وضعية relative للأب.

11. Responsive Design & Media Queries

Responsive design means your site looks good on mobile, tablet, and desktop.

@media (max-width: 768px) {{ .nav {{ flex-direction: column; }} }}

Exercise

أضف media query يجعل عناصر الدروس تظهر في عمود واحد عندما يكون عرض الشاشة صغير.

12. Buttons & Hover Effects

Buttons can be styled with background, border-radius, and hover transitions.

.btn-primary {{ background: linear-gradient(135deg,#f97316,#facc15); border-radius: 999px; padding: 10px 18px; transition: transform 0.2s; }} .btn-primary:hover {{ transform: translateY(-2px); }}

Exercise

صمّم زر "ابدأ التعلم" مع تأثير hover يرفع الزر قليلاً ويغيّر لون الخلفية.

13. Transitions & Animations

Use transitions for smooth changes and keyframes for more complex animations.

.button {{ transition: background 0.3s ease; }} .button:hover {{ background: #22c55e; }}

Exercise

أضف انتقال (transition) على رابط في القائمة بحيث يتغير اللون تدريجياً عند المرور عليه.

14. CSS Variables

CSS variables let you reuse values like colors and spacing across your styles.

:root {{ --main-color: #f97316; }} h1 {{ color: var(--main-color); }}

Exercise

أنشئ متغير لون أساسي في :root واستخدمه في خلفية الهيدر والأزرار.

15. Mini Project: Simple Landing Page

Combine your HTML and CSS skills to build a small one-page website.

  • Hero section with title, text, and button.
  • Features section using flexbox or grid.
  • Footer with contact information.

Exercise

صمّم صفحة ترحيبية لمشروع "شبكة إنترنت" فيها قسم للخدمات، الأسعار، وطريقة التواصل.
