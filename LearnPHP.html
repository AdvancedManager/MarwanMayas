1. What is PHP?

PHP is a server-side language used to build dynamic and interactive websites.

<?php echo "Hello PHP!"; ?>

Exercise

اكتب كود يعرض جملة ترحيبية بالمستخدم.

2. PHP Syntax

Every PHP script begins with <?php and ends with ?>.

<?php $name = "Ali"; echo $name; ?>

Exercise

أنشئ متغير يحتوي اسمك وقم بطباعته.

3. Variables

Variables in PHP always start with the $ symbol.

$age = 20; $city = "Sanaa";

Exercise

اكتب 3 متغيرات (الاسم – العمر – المدينة).

4. Data Types

PHP supports strings, numbers, booleans, arrays, and objects.

$name = "Mohammed"; $marks = [90, 80, 70]; $isActive = true;

Exercise

كوّن مصفوفة تحتوي على 5 مواد دراسية.

5. Strings

Strings can be concatenated using the . operator.

$fullName = "Ali" . " Mohammed"; echo $fullName;

Exercise

ادمج اسمك الأول والأخير باستخدام .

6. Arrays

Arrays store multiple values in one variable.

$fruits = ["Apple","Banana","Mango"]; echo $fruits[1];

Exercise

اطبع ثاني عنصر من مصفوفة مكونة منك.

7. Associative Arrays

Associative arrays hold key-value pairs.

$user = [ "name" => "Ali", "age" => 21 ]; echo $user["name"];

Exercise

أنشئ مصفوفة تحتوي (الاسم – العمر – الجامعة).

8. Conditions

Conditions allow PHP to make decisions.

if ($age >= 18) { echo "Adult"; } else { echo "Minor"; }

Exercise

اكتب كود يتحقق هل الطالب ناجح أو راسب حسب الدرجة.

9. Loops

PHP supports for, while, and foreach loops.

for ($i=1; $i<=5; $i++) { echo $i; }

Exercise

اطبع الأرقام من 1 إلى 10 باستخدام loop.

10. Functions

Functions group reusable code.

function greet($name) { return "Hello " . $name; } echo greet("Yemen");

Exercise

أنشئ function يرجع مربع رقم.

11. Superglobals

Superglobals like $_GET and $_POST store form data.

echo $_GET["user"];

Exercise

اعرض قيمة في الرابط باستخدام GET.

12. Form Handling

PHP handles input submitted from HTML forms.

if ($_SERVER["REQUEST_METHOD"]=="POST") { echo $_POST["name"]; }

Exercise

صمّم فورم بسيط يعرض الاسم بعد الضغط.

13. Include & Require

Used to import reusable files.

include "header.php";

Exercise

أنشئ ملف مشترك header واستخدمه في صفحة ثانية.

14. Connecting to MySQL (Basics)

Use MySQLi or PDO to connect to a database.

$conn = new mysqli("localhost","root","","school");
