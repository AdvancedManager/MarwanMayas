1. What is JavaScript?

JavaScript is a programming language that runs inside the browser and controls interactive behavior.

console.log("Hello JavaScript!");

Exercise

اطبع جملة باسمك في console باستخدام console.log().

2. Variables (let, const, var)

Variables store data for usage later in the script.

let name = "Ali"; const age = 20; var city = "Sanaa";

Exercise

أنشئ ثلاث متغيرات: الاسم، العمر، والهواية.

3. Data Types

JavaScript has numbers, strings, booleans, arrays, objects, and more.

let isStudent = true; let marks = [90, 85, 70]; let person = {{ name: "Ali", age: 21 }};

Exercise

أنشئ object يحتوي على اسمك وعمرك وقسمك الدراسي.

4. Operators

JS supports arithmetic, comparison, and logical operators.

let sum = 5 + 3; let check = (10 > 3) && (2 < 5);

Exercise

اكتب معادلة تجمع رقمين وتطبع الناتج في console.

5. Conditions (if, else)

Conditions allow your code to make decisions.

if (age >= 18) {{ console.log("Adult"); }} else {{ console.log("Minor"); }}

Exercise

اكتب برنامج يخبرك هل الطالب ناجح أو راسب حسب درجته.

6. Loops (for, while)

Loops repeat code multiple times.

for (let i = 1; i <= 5; i++) {{ console.log("Count:", i); }}

Exercise

أنشئ loop يطبع الأرقام من 10 إلى 1.

7. Functions

Functions group reusable code.

function greet(name) {{ return "Hello " + name; }}

Exercise

8. Arrays

Arrays store multiple values.

let fruits = ["Apple", "Orange", "Banana"]; console.log(fruits[1]);

Exercise

كوّن مصفوفة فيها 5 مواد دراسية.

9. Objects

Objects store data in key-value pairs.

let laptop = {{ brand: "HP", cpu: "i9" }}; console.log(laptop.cpu);

Exercise

أنشئ object يحتوي على معلومات جهازك الشخصي.

10. DOM Manipulation

DOM allows JS to control HTML elements.

document.getElementById("title").textContent = "Changed!";

11. Events

Events detect user actions like clicks or typing.

button.onclick = function() {{ alert("Clicked!"); }};

12. Forms & Validation

JS validates input before sending it to the server.

if (nameInput.value === "") {{ alert("Name required"); }}

13. JSON Basics

JSON is a format for storing structured data.

let text = '{{ "name": "Ali", "age": 20 }}'; let obj = JSON.parse(text);

14. Local Storage

Used for saving data inside browser.

localStorage.setItem("theme", "dark");
