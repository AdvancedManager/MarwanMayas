Learn Web Development From Zero

This educational website is your simple guide to start learning front-end and back-end web development: HTML for structure, CSS for design, JavaScript for interaction, and PHP for dynamic websites.

Start with HTML About the Devloper

HTML Basics

Learn how to build the structure of web pages using tags, elements, and attributes.

CSS Styling

Control colors, fonts, layout, and make your pages look modern and professional.

JavaScript Logic

Add interactivity, events, validation, and dynamic behavior to your pages.

PHP & Backend

Connect your website to the server, forms, databases, and build real web applications.

What will the student learn?

This project is designed as a simple training platform for web development. The student can browse lessons, practice with examples, and solve quizzes in Arabic-friendly style with English code.

Track 1 – HTML

Everything about web page structure: headings, paragraphs, links, images, tables, forms, and semantic tags.

Includes

  • ~15 lessons
  • Exercises & examples
  • HTML quiz + final exam

Track 2 – CSS

Learn how to style your pages using colors, fonts, layout, flexbox, grid, and responsive design.

Includes

  • ~15 lessons
  • Practice tasks
  • CSS quiz + final exam

Track 3 – JavaScript

Programming in the browser: variables, conditions, loops, functions, DOM, events, and simple projects.

Includes

  • ~15 lessons
  • Interactive examples
  • JS quiz + final exam

Track 4 – PHP

Back-end basics: PHP syntax, forms, sessions, simple login, and connecting to a database (theoretically).

Includes

  • ~15 lessons
  • Server-side exercises
  • PHP quiz + final exam
