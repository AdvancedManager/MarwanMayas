HTML Lessons

1. What is HTML?

HTML stands for HyperText Markup Language. It is used to describe the structure of web pages.

HTML is not a programming language, it is a markup language that uses tags to tell the browser what each part means.

<!DOCTYPE html> <html> <head> <title>My first page</title> </head> <body> <h1>Hello, HTML!</h1> </body> </html>

Exercise

Create a new HTML file and write a simple page with a title and one heading that contains your name.

2. Basic Page Structure

An HTML document has three main parts: <html>, <head>, and <body>.

The <head> contains information about the page (title, meta tags, links). The <body> contains the visible content.

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Student Page</title> </head> <body> <p>This is my first web page.</p> </body> </html>

Exercise

أضف فقرتين جديدتين في <body>، الأولى تعريف بسيط عن نفسك، والثانية هواياتك.

3. Headings & Paragraphs

Headings go from <h1> (largest) to <h6> (smallest). Paragraphs use <p>.

<h1>Main Title</h1> <h2>Section Title</h2> <p>This is a normal paragraph of text.</p>

Exercise

صمّم صفحة فيها عنوان رئيسي h1 لاسم الموقع، وعناوين فرعية h2 للأقسام (Home, Courses, About).

4. Links & Navigation

Use <a> tags to create links between pages or to external sites.

<a href="https://example.com">Visit Example</a> <a href="about.html">About Us</a>

Exercise

أنشئ صفحتين: index.html وabout.html، واربط بينهما باستخدام الروابط.

5. Images

Use the <img> tag to display images. Always add the alt attribute for accessibility.

<img src="images/logo.png" alt="Site Logo" width="120" height="120">

Exercise

ضيف شعار مشروعك في الصفحة الرئيسية، واكتب نص بديل يصف الشعار لو ما اشتغلت الصورة.

6. Ordered & Unordered Lists

HTML has ordered lists (<ol>) and unordered lists (<ul>).

<ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul> <ol> <li>Step 1</li> <li>Step 2</li> </ol>

Exercise

اعمل قائمة unordered لهواياتك، وقائمة ordered لخطوات تثبيت برنامج المتصفح.

7. Tables

Tables organize data in rows and columns using <table>, <tr>, <th>, and <td>.

<table border="1"> <tr> <th>Name</th> <th>Grade</th> </tr> <tr> <td>Ali</td> <td>A</td> </tr> </table>

Exercise

صمّم جدول لنتائج ثلاثة طلاب في مادة "Web Applications" يحتوي على الاسم والدرجة.

8. Forms & Inputs

Forms allow users to send data to the server using fields like text, email, password, and buttons.

<form action="register.php" method="post"> <label>Name:</label> <input type="text" name="name"><br> <label>Email:</label> <input type="email" name="email"><br> <button type="submit">Register</button> </form>

Exercise

اعمل نموذج تسجيل دخول يحتوي على حقل username و password وزر Login فقط.

9. Attributes & Links Target

Attributes provide extra information to elements, like href, src, alt, title, and id.

<a href="https://example.com" target="_blank" title="Open new tab"> Open site </a>

Exercise

جرّب إضافة <a> يفتح موقع جامعتك في تبويب جديد باستخدام الخاصية target="_blank".

10. Div & Span

<div> is a block-level container, <span> is inline. They help you group content for styling.

<div class="card"> <h2>Course Title</h2> <p>Short description here</p> </div>

Exercise

استخدم <div> لتجميع عنوان، صورة، وفقرة عن مادة "تطبيقات الويب" وكأنها كرت واحد.

11. Semantic HTML

Semantic tags describe the meaning of content: <header>, <nav>, <main>, <section>, <footer>.

<header>Site Header</header> <nav>Main Menu</nav> <main>Page Content</main> <footer>Footer Info</footer>

Exercise

أعد كتابة صفحتك الرئيسية باستخدام <header>, <nav>, <main>, <footer> بدل <div> فقط.

12. Audio & Video

You can embed audio and video using <audio> and <video> tags.

<video width="320" controls> <source src="intro.mp4" type="video/mp4"> Your browser does not support the video tag. </video>

Exercise

أضف فيديو تعليمي قصير عن HTML في صفحتك مع أزرار التحكم (تشغيل/إيقاف).

13. Comments

Comments help you write notes inside your code without showing them in the browser.

<!-- This is a comment, it will not appear on the page --> <p>Visible text</p>

Exercise

أضف تعليق في أعلى كل صفحة يوضح اسم الطالب، المادة، واسم الدكتور.

14. Best Practices

Write clean, readable HTML so that other developers (and your future self) can understand it easily.

  • Use lowercase tag names and proper indentation.
  • Use semantic tags instead of too many generic <div> elements.
  • Add meaningful alt text to images.
  • Validate your HTML using an online validator.

  • Use header, nav, main, and footer.

  • Use headings, paragraphs, lists, images, and a simple form.
